# 清页 / Tab Deck

一个离线可安装的 Chrome 新标签页扩展。

打开新标签页后，它会把当前所有网页标签按站点分组，方便你：

- 快速跳转到已打开标签
- 关闭单个标签 / 整组标签 / 全部标签
- 清理重复标签
- 把标签暂存到 **稍后处理**
- 通过搜索快速定位已开页面或直接打开网址

## 特点

- **本地运行**：不依赖后端服务
- **离线可测试**：把 `extension/` 文件夹发给别人即可安装
- **新标签页替换**：打开新标签页时直接进入 Tab Deck
- **中英双语文案**：会根据浏览器语言自动切换

## 当前包含的主要能力

- 接管 Chrome 新标签页
- 按域名聚合当前标签页
- 重复标签提示与一键清理
- 行内快速操作（稍后 / 关闭 / 重新打开 / 移除）
- 快捷入口（Quick Links）
- 搜索已打开标签，也支持直接输入网址打开
- “全部关闭”轻量确认弹窗

## 安装方式（给测试者）

### Chrome / Edge

1. 解压压缩包
2. 打开浏览器扩展页：
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
3. 打开右上角 **开发者模式**
4. 点击 **加载已解压的扩展程序** / **Load unpacked**
5. 选择压缩包里的 `extension/` 文件夹

> 注意：不是选择整个 `qingye/`，而是里面的 `extension/` 目录。

## 隐私说明

这个扩展不需要登录，不连接后端，不上传标签数据。

使用到的数据仅保存在浏览器本地：

- `chrome.storage.local`
- 当前浏览器标签页信息

## 测试建议

可以重点帮忙看这几块：

- 新标签页首屏观感
- 分组是否自然
- 搜索是否顺手
- 重复标签提示是否清楚
- 行内按钮是否容易理解
- 中英文文案是否顺
